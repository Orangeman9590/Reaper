# Reaper
Find out the geo-location of an IP adress.
------------
WORKS IN ANY OS
----------------
HOW TO USE
---------
Download repo(Windows) or Clone Repo(Linux/Unix)
----------------
```
$ git clone https://github.com/Orangeman9590/Reaper
```
-------------
Change Directories
-----------
```
$/C: cd Reaper
```
-----------
Install Requirements
-----------
```
$/C: pip3 install -r requirements.txt
```
-------------
Run
----------
```
$/C: python3 reaper.py
```
--------
